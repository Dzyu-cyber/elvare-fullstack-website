# Elvaré Fullstack Website

A production-grade fashion e-commerce platform built with Next.js, NestJS, and PostgreSQL.

## Project Structure

- `apps/web`: Next.js frontend
- `apps/api`: NestJS backend

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion, Zustand, TanStack React Query
- **Backend**: NestJS, Prisma, PostgreSQL, JWT Auth
- **Other**: pnpm workspaces

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env` and fill in the values.

3. Run development servers:
   ```bash
   pnpm dev
   ```
